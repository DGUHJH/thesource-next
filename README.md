# Thesource-next

## Development Stack
- Vercel
- Next.js
- Typescript
- Redux

## Installation
- install modules
```
npm install
```

## Test
- localhost: 3000
```
npm run dev
```

## Deploy
- linked vercel
```
$ git push origin develop:master
```

## Naming Convention
1. class
- Pascal case / TextField
2. function, variant
- Camel case / textField
3. css, database, cookies, etc...
- Snake case / text_field
